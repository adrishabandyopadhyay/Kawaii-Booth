# Kawaii-Booth 🎀

A browser-based photo print studio that lets users upload photos, choose layouts, and generate print-ready A4 sheets.

## 🚀 Features
- Upload and preview multiple images
- Choose layouts (1, 2, 3, or 4 photos per sheet)
- Drag-and-drop photo reordering
- Client-side image editing (crop, adjust, preview)
- Export high-quality PDF using jsPDF
- No backend required (fully frontend-based)

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Canvas API
- jsPDF

## 📌 How It Works
1. Upload your photos  
2. Select layout and styling  
3. Customize arrangement  
4. Download print-ready PDF  

## 💡 Notes
This project is designed as a lightweight, self-contained web tool demonstrating frontend logic, UI handling, and client-side processing.
