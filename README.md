# Kawaii-Booth 🎀

A frontend photo booth web application that allows users to upload, arrange, and export photos into print-ready A4 layouts.

## Features
- Upload multiple images
- Choose layout (1, 2, 3, or 4 photos)
- Drag-and-drop arrangement
- Live preview
- Export as PDF

## Tech Stack
- HTML
- CSS
- JavaScript
- Canvas API
- jsPDF

## Usage
Upload photos → select layout → customize → download PDF
